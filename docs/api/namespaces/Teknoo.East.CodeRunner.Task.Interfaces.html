\Teknoo\East\CodeRunner\TaskInterfaces

Interfaces

CodeInterface Interface to define values objects to store code to execute. These value object can also required several capabilities like a specific platform, packages, or extensions, returned by the method getNeededCapabilities.
ResultInterface Interface to define, as value object, the result for task.
StatusInterface Interface to define value object to define a status of a task.
TaskInterface Interface to define task class, able to trace the process under all service.
