\Teknoo\East\CodeRunner\Task\Interfaces

\Teknoo\East\CodeRunner\Task\InterfacesCodeInterface

Interface to define values objects to store code to execute. These value object can also required several capabilities like a specific platform, packages, or extensions, returned by the method getNeededCapabilities.
