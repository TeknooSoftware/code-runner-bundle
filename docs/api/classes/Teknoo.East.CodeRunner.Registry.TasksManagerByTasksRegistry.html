\Teknoo\East\CodeRunner\RegistryTasksManagerByTasksRegistry

Class TasksManagerByTasksRegistry.

Default implementation of TasksManagerByTasksRegistryInterface to persist the task manager managing a task. The registry is usable via an array access behavior, with tasks as key, to return and manipulate managers. Manager are identified by their id referenced in the platform, but manager must be referenced into the registry. The registry use TaskRegistration entity to persist and manage the relation.
