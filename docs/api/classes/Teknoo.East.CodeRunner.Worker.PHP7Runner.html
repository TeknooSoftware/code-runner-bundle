\Teknoo\East\CodeRunner\WorkerPHP7Runner

Class PHP7Runner.

Implementation of RunnerInterface, Worker of RemotePHP7Runner of CodeBundle library to execute some PHP Task into an isolated and secured environment. Tasks and returns are transmitted via two dedicated AMQP exchange.
