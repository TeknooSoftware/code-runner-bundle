\Teknoo\East\CodeRunner\EndPointLoadNextTasksEndPoint

Class LoadNextTasksEndPoint.

End point, used by East Foundation to allow developer to create a cron request to ask platfrom to switch to next stand by tasks when a runner idle and are not automatically switched to new task.
