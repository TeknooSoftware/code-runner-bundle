\Teknoo\East\CodeRunner\Entity\Task

\Teknoo\East\CodeRunner\Entity\TaskTask

Class Task.

Default implementation of TaskInterface, using States pattern with the Teknoo States library to manage each state and status of a task in this library workflow. States are automatically applied with the states Lifecycable extension.
