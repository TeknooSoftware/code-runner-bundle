\Teknoo\East\CodeRunner\RegistryTasksByRunnerRegistry

Class TasksByRunnerRegistry.

Default implementation of TasksByRunnerRegistryInterface to persist the task currently executed by a runner. The registry is usable via an array access behavior, with runners as key, to return and manipulate tasks. The registry use TaskExecution entity to persist and manage the relation.
